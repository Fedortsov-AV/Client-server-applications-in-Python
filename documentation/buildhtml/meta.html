class meta.ClientVerifier(clsname, bases, clsdict)¶

Базовые классы: type

Метокласс клиента, проверяет отсутствие вызова метода accept, а так же использование TCP (по наличию константы SOCK_STREAM при создании сокета)

class meta.ServerVerifier(clsname, bases, clsdict)¶

Базовые классы: type

Метокласс сервера, проверяет отсутствие вызова метода connect, а так же использование TCP (по наличию константы SOCK_STREAM при создании сокета)
