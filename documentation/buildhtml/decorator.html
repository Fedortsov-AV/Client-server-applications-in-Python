decorator module¶

decorator.login_required(func)¶

Декоратор, проверяющий авторизацию клиента на сервере. Пропускает PRESENCE, AUTH, REG сообщения. Для остальных сообщений проверят наличие сокета клиента в dict подключенных к серверу пользователей.

decorator.logs(func)¶

Декоратор для вывода в лог сообщения с указанием функции вызвавшей декарируемую функцию

decorator.verify_edit(func)¶

Декоратор находит все поля типа *Edit и проверяет данные в них

Результат

В случае успеха возвращает результат декорируемой функции.

Если поля пустые или содержат недопустимые данные возвращает QMessageBox
