common.utils module¶

Модуль хранит общие функции для сервера и клиента

common.utils.get_message(sock: socket) → dict¶

Функция для получения данных из сокета и преобразования полученных байтов в dict :param sock: сокет из которого получаем данные :return: При удачном получении данных возвращает dict c полученным данными. При исключениях TypeError или ValueError возвращает словарь: ANS_400 = { ACTION: ALERT, RESPONSE: 400, ALERT: „Неправильный запрос/JSON-объект“, TIME: time.time() } При исключениях ConnectionResetError, OSErrorr возвращает словарь: ANS_104 = { ACTION: ALERT, RESPONSE: 104, ALERT: „ConnectionResetError“, TIME: time.time() }

common.utils.send_message(msg: dict, sock: socket) → bool¶

Функция для отправки сообщения, создает JSON из dict, кодирует его в байты и отправляет в адрес сокета

Параметры

  • msg – dict для создания JSON

  • sock – сокет, в который необходимо отправить байты

Результат

Возвращает True, если отправка успешна, либо False если произошла ошибка
